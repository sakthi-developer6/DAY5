# Day 5 – FastAPI Validation & Project Structure

This project was developed as part of the Software Development Internship – Day 5 task. The objective was to build a professional backend application using FastAPI by following a clean and scalable project structure.

## Features

* FastAPI REST API
* SQLAlchemy ORM integration
* MySQL Database Connection
* Pydantic Models for Input Validation
* HTTP Status Codes
* Student CRUD Operations
* Modular Project Structure
* Routers, Models, Schemas, Services, and Database Layers
* Swagger API Documentation
* Clean Code Architecture following backend development best practices

## Project Structure

* `app/database` – Database connection and session management
* `app/models` – SQLAlchemy ORM models
* `app/schemas` – Pydantic request and response models
* `app/services` – Business logic implementation
* `app/routers` – API endpoints
* `app/main.py` – FastAPI application entry point

## Technologies Used

* Python
* FastAPI
* SQLAlchemy
* MySQL
* Pydantic
* Uvicorn

This project demonstrates professional backend development practices, including modular architecture, data validation, database integration, and maintainable code organization.
